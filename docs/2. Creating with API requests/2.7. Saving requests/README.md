# Сохранение запросов

- Сохранение запроса с помощью коллекции
- Коллекции помогают организовать запросы
- Также можно добавить документацию, тесты и все, что является полезным при расшаривании с командой

Для сохранения просто нажимаем кнопку "Save" рядом с "Send".

Можно расшаривать коллекции с другими членами команды:

- Экспорт коллекции как файла, импорт файла
- Расшаривание по ссылке (нужен аккаунт Postman)
- Встраиваемая кнопка (нужен аккаунт Postman)
- Расшаривание с командой (нужен Postman Pro)

> Коллекции и запросы сохраняются и синхронизируются с Postman аккаунтом
